# Title
Medical Chatbot (FastAPI)

# Description
This is a simple medical chatbot built using FastAPI. 
It takes user symptoms as input and returns basic health-related responses.

# Features
Chat-based interaction
Symptom-based response
FastAPI backend
Simple and easy to use

# Technologies Used
Python
FastAPI
MySQL
Postman (for testing)

# How to Run
# Clone project
git clone https://github.com/yourusername/medical-chatbot-.git

# Go to folder
cd project folder

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn main:app --reload

# Note
This chatbot provides only general information and is not a replacement for professional medical advice.
